# 🚗 Автосалон JETOUR 

## 📌 О проекте
Проект разработан в рамках учебной практики для автоматизации учёта продаж официального дилера **JETOUR**.

Система включает три программных продукта:
- **Десктопное приложение** (Python + Tkinter) — для работы персонала автосалона
- **Мобильное приложение** (Kotlin + XML) — для менеджеров и клиентов
- **Сайт-лендинг** (HTML + CSS + JS) — для скачивания приложений

## 🚀 Функциональные возможности

### Десктопное приложение
- ✅ Управление автопарком (добавление, редактирование, удаление, поиск, сортировка)
- ✅ Оформление продаж с автоподсчётом скидки (5% за кредит)
- ✅ Ведение клиентской базы
- ✅ Запись на тест-драйв
- ✅ 5 видов отчётов с сохранением в TXT

### Мобильное приложение
- ✅ Просмотр автомобилей 
- ✅ Оформление продаж
- ✅ Запись на тест-драйв
- ✅ Клиентская база
- ✅ Статистические отчёты

### Сайт-лендинг
- ✅ Описание приложений
- ✅ Скачивание файлов (.exe и .apk)
